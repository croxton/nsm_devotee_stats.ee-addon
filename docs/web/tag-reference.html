{exp:nsm_devotee_stats:fetch}

([
    ...

])}

Parameters

addon

addon="nsm-devotee-stats"

Enter the url-title of your add-on as it appears on Devot:ee. This is the second segment of your Devot:ee add-on details page URL.

Single variables

{title}

{title}

The name of the add-on as it appears on Devot:ee.

{developer_name}

{developer_name}

The name of the add-on developer.

{developer_alt}

{developer_alt}

Alternative developer name for the add-on.

{license}

{license}

License type of the add-on. Either Free or Commercial.

{party}

{party}

Type of add-on for ExpressionEngine. Either 1st Party or 3rd Party.

{version_support_ee1}

{version_support_ee1}

Boolean value for add-on's support of ExpressionEngine 1.x.

{version_support_ee2}

{version_support_ee2}

Boolean value for add-on's support of ExpressionEngine 2.x.

{summary}

{summary}

The add-on summary.

{description}

{description}

The add-on description.

{rating_percentage}

{rating_percentage}

The percentage value (0-100) of the add-on's rating on Devot:ee.

{rating_rounded}

{rating_rounded}

The add-on rating rounded to a single decimal place (0.0-5.0).

{rating}

{rating}

The raw value (0.0000-5.0000) of the add-on's rating on Devot:ee.

{votes}

{votes}

Total Devot:ee user votes for add-on.

{favorites}

{favorites}

Total number of times that this add-on has been set as a favourite by Devot:ee users.
