Requirements

Technical requirements include:

  • ExpressionEngine 2.2+
  • PHP 5.2+
  • A modern browser: Firefox, Safari, Google Chrome or IE8+

Installation

  1. Download the latest version of NSM Devot:ee Stats
  2. Extract the .zip file to your desktop
  3. Copy system/expressionengine/third_party/nsm_devotee_stats to system/expressionengine/third_party
  4. Activate NSM Devot:ee Stats in the ExpressionEngine Control Panel
